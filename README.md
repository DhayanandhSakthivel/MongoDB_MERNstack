# House Renting Application

A full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) that allows users to browse, list, and manage house rentals. This application provides functionality for both renters and owners, including property search, listing management, and user authentication.

## Features

- **User Authentication**: Sign up and log in with role-based access for renters and owners.
- **Property Listing**: Owners can add, update, and delete house rental listings.
- **Property Search**: Renters can search for properties based on location, price range, and other filters.
- **Booking System**: Renters can book houses directly from the application.
- **Admin Dashboard**: Admin can manage users and listings.
- **Responsive Design**: Mobile-first design ensures the application works across various screen sizes.
- **Secure API**: Built-in security features for the API endpoints (JWT Authentication).

## Technologies Used

- **MongoDB**: Database for storing user data, house listings, and booking information.
- **Express.js**: Web framework used to handle the backend routes and APIs.
- **React.js**: Frontend framework used to build an interactive and dynamic user interface.
- **Node.js**: Runtime environment to build the backend of the application.
- **JWT**: JSON Web Tokens for secure user authentication.
- **Mongoose**: MongoDB object modeling tool to manage data schema.
- **Axios**: HTTP client for making API requests from the frontend.
